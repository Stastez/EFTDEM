heightMap * RasterizerGPU::apply ( pointGrid *  pointGrid,
bool  generateOutput 
)
overridevirtual

Use the currently attached ICloudRasterizer to convert a pointGrid to a heightMap.

Parameters
pointGridThe sorted pointGrid to be read from. If the current ICloudRasterizer uses OpenGL, the points component is optional if a previous stage already set the appropriate buffers on the GPU
generateOutputWhether to return a fully formed heightMap (including the actual height data for each pixel). This may be set to false if the following stage uses OpenGL and can employ the already set buffers on the GPU
Returns
A fully formed heightMap if generateOutput is true, a heightMap only containing all necessary metadata otherwise

Implements ICloudRasterizer.
