◆ initializeGL()

GLFWwindow * GLHandler::initializeGL ( bool  debug = false)

Initializes the GLHandler by creating a GLFW OpenGL context as well as a glbinding context. This will also generate all buffers contained in GLHandler::bufferIndices and set up buffer management. The program may exit with EXIT_OPENGL_ERROR for any of these reasons:

  • GLFW could not initialize
  • GLFW could not create a window with the specified parameters
Parameters
debugWhether or not to create a debug OpenGL context. Usually you want this to be false.
Returns
A pointer to the created GLFWwindow (i.e., the OpenGL context)
