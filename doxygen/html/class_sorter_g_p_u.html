Use the currently attached ICloudSorter to sort the given point cloud into a pointGrid of the previously specified resolution.

Parameters
pointCloudThe rawPointCloud to be sorted.
generateOutputWhether to return a fully formed pointGrid (including the actual points). This may be set to false if the following stage uses OpenGL and can employ the already set buffers on the GPU
Returns
A fully formed pointGrid if generateOutput is true, a pointGrid only containing all necessary metadata otherwise
